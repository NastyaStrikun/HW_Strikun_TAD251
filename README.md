# Описание проекта
В данном проекте представленна реализация классов, необходимых для управления рецептами (создание рецептов, добавление ингредиентов, контроль над порциями и пр)
# Установка
Для установки необходимо склонировать репозиторий
Также установить содержимое requirements.txt
```bash
git clone <ссылка-на-репозиторий>
cd HW_Strikun_TAD251
pip install -r requirements.txt 
```

# Тесты
Проект протестирован, тесты находятся в отдельном файле
```bash
pip install -r requirements.txt
paytest
```
